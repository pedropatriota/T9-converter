# T9 converter

This is a [React](https://reactjs.org) + [TypeScript](https://www.typescriptlang.org/) boilerplate built with [Vite](https://vitejs.dev).

## Getting started

1. Intall the dependencies: 
```bash
  npm install
  ```
  
2. Run client dev (port 3000)
```bash
  npm run dev 
  ```
  
3. Server dev (port 3333)
```bash
  npm run server:dev
  ```
